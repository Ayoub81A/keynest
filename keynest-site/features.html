Features

Built for real security

Every feature in KeyNest is designed with one principle: your data belongs to you, and only you.

Password Vault

Store every password, organised and secure

Your entire digital life in one place. Every password is encrypted before it leaves your browser so the server stores only ciphertext it can never decode.

  • Add, edit, and delete entries with site name, username, URL, and password
  • One-click copy for both password and username
  • Show or hide passwords individually per entry
  • Real-time password strength indicator (Weak / Medium / Strong)
  • Instant search across all entries

Generator

Generate passwords you'll never have to remember

Create cryptographically random passwords up to 40 characters long. Adjust the character set to match any site's requirements, then save directly to your vault in one click.

  • Cryptographically random using the browser's Web Crypto API
  • Configurable length from 8 to 40 characters
  • Toggle uppercase, lowercase, numbers, and symbols
  • "Use this password" opens a pre-filled new entry form

Health Report

Know exactly how secure your vault is

KeyNest analyses every password in your vault and gives you a health score out of 100. See weak, reused, and medium passwords all in one place — and fix them with one click.

  • Vault health score calculated from weak, reused, and medium passwords
  • Identifies passwords used on more than one site
  • One-click Edit to fix any problem entry immediately

Secure Notes

Not everything sensitive is a password. Store WiFi credentials, license keys, PINs, API keys, or any private text as an encrypted note — with a distinct visual style so notes are always easy to spot.

Trash & Restore

Deleted entries don't disappear immediately. They move to a trash bin and are kept for 30 days, giving you time to restore anything deleted by mistake. After 30 days they're permanently purged.

The details matter too

Custom Tags
Create tags like Work, Personal, or Finance and filter your vault instantly.
One-Click Copy
Copy passwords or usernames to your clipboard instantly with visual confirmation.
Instant Search
Search across all entries by title or username in real time as you type.
Secure Sessions
Sessions expire when you close your browser tab. The encryption key is wiped from memory on logout.
Mobile Friendly
Fully responsive with a mobile bottom navigation and slide-in drawer for a native app feel.
Zero Knowledge
The server stores only ciphertext. Your encryption key is never transmitted, stored, or logged.
